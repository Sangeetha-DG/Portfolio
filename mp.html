Math Processing Server (MP)

Technologies: Java, Socket Programming, HTTP Protocol, Expression Parsing

Abstract

Server Architecture Diagram

This project implements a custom TCP server application written in Java that listens on a configurable port, parses mathematical expressions sent via HTTP requests, validates and computes the result, and responds with the output following a lightweight protocol over HTTP.

Introduction

The Math Processing Server (MP) is designed to process arithmetic and trigonometric expressions received from clients through HTTP requests. The server parses the expressions, validates syntax including operator precedence and balanced parentheses, performs the calculations, and returns results in a structured HTTP response.

Technical Implementation

The server is implemented using Java's networking libraries, specifically the ServerSocket and Socket classes, to accept client connections and read input streams. Expressions are parsed byte-by-byte, decoded from URL format, and validated against allowed operators and syntax rules. Mathematical operations include addition, subtraction, multiplication, division, exponentiation, and trigonometric functions (sin, cos, tan).

The server sends raw HTTP response headers followed by the calculated result. Input validation ensures that malformed or invalid expressions return appropriate error messages.

Challenges and Learnings

Key challenges included handling concurrent client connections, performing accurate parsing and evaluation of mathematical expressions, and formatting correct HTTP responses. This project enhanced understanding of raw socket communication, HTTP fundamentals, and complex string parsing in Java.

Future Work

Future improvements could involve implementing a full RESTful API, expanding supported mathematical functions, improving concurrency with multithreading, and enhancing input sanitization for security.
