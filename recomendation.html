YouTube Video Recommendation using Twitter API

Home/ Next Project →

Problem statement: In this project, we are building a recommendation system that suggests transcribed video content matching Twitter topic categories. You will select 10 Twitter users and determine which YouTube transcriptions are relevant for each user, based on their past retweets.

Description:
• Used the Twitter API to fetch user data and YouTube Data API to collect video titles.
• Built a cross-platform system that recommends YouTube videos based on trending topics from Twitter.
• Django was used as the web framework to manage the backend and API integration.
• Extracted top news topics from users’ tweets in real-time to match relevant YouTube videos.

Cross-Platform Data Analysis:

  • Collected data from Twitter and YouTube manually using known trending topics.
  • Used TF-IDF (via TfidfVectorizer from scikit-learn) to represent and compare tweet and video content.
  • Applied cosine similarity to compute the most relevant YouTube video recommendations.
