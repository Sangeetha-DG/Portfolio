YouTube Video Recommendation using Twitter API

Cross-Platform Topic Modeling & Content Ranking | By Sangeetha D G

Abstract

This project proposes a recommendation system leveraging Twitter user behavior and YouTube video transcripts. Using trending tweets and user retweets, it maps relevant videos based on content similarity. The system integrates both platforms to demonstrate how text mining can drive content discovery.

Project Metadata

  • Category: Data Mining
  • Client: Academic
  • Date: 20 May 2023
  • Project Repository: GitHub Link

System Overview

  • Used Twitter API to extract user tweets and engagement history.
  • Fetched video titles from YouTube Data API.
  • Django was implemented for backend logic and API management.
  • Matched videos to users by applying text mining on tweet content and transcripts.

Recommendation Pipeline

  • Topic extraction from tweets using TF-IDF vectorization.
  • Representation of both tweet and video transcript data.
  • Applied cosine similarity to identify closely matching video titles.
  • Top 5 recommended videos returned based on each Twitter profile.

Conclusion

This thesis-style implementation showcases how user-generated content from Twitter can guide automated video recommendations. The blend of APIs, machine learning, and NLP helps connect user interest to content relevance in real-time.
